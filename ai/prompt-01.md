# AI Waiter

Use LangGraph samples in [lang-graph-sample](lang-graph-sample) as template

Need a chatbot AI waiter for a restaurant providing a tray/shopping-cart and taking orders.  tray, orders can be in-memory data structures in python

Restaurant food menu is found as [menu.json](../menu/menu.json)

Need a simple local desktop program that uses LangGraph running in a loop.

The chat-bot app needs to greet user, ask how the user may be helped.
Based on user input based on his conversation in the chat prepare the tray with quantities.  Once user confirms place the order.  Simulate an order status of placed / preparing / ready / served after few minutes of placement order.
